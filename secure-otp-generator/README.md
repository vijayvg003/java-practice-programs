# Secure OTP Generator

This Java program generates a secure One-Time Password (OTP) using the SecureRandom class.

## Features
- Generates numeric OTP
- Uses SecureRandom for security
- Customizable OTP length

## Technologies Used
- Java
- SecureRandom
