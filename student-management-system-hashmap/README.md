# Student Management System (Java)

This project demonstrates a simple Student Management System using the Java Collection Framework.

## Features
- Add student
- View students
- Search student
- Delete student
- Menu-driven console program

## Technologies Used
- Java
- HashMap (Collection Framework)
- OOP Concepts
- Scanner class
